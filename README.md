# E-Commerce Web Application

## Technologies Used
- **React.js (Vite)**
- **Redux Toolkit & Redux Saga** (State Management)
- **React Router** (Navigation)
- **Axios** (API Requests)
- **Formik & Yup** (Form Validation)
- **Pure CSS** (Styling)
- **DummyJSON API** (Mock Backend)

## Setup Instructions

**Step 1. Clone the project code to your local machine:**
```bash
git clone https://github.com/54321deepak/frontend-technical-assignment.git
cd frontend-technical-assignment
```

**Step 2. Install all the necessary packages:**
```bash
npm install
```

**Step 3. Start the application:**
```bash
npm run dev
```

That's it! Open your browser and go to `http://localhost:5173` to view the app.
